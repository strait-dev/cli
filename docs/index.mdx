## Plays nice with your tools

Every command outputs table for humans and JSON for machines. Pipe to `jq`, feed into scripts, or use in CI — the CLI handles both:

```bash
strait jobs list                    # Pretty table in your terminal
strait jobs list -o json | jq '.'  # Clean JSON for scripts
```

<Info>
  Styled output goes to stderr. Data goes to stdout. So you can pipe JSON while still seeing progress messages — no parsing tricks needed.
</Info>

Six output formats built in: `table`, `json`, `yaml`, `csv`, `go-template`, `jsonpath`.
