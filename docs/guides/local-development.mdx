---
title: "Local Development"
description: "Route signed Strait pushes to a local endpoint"
---

Run your application locally with its normal framework command, then use
`strait dev` to make Strait reach it through a temporary tunnel.

```bash
npm run dev
strait dev --port 3000
```

`strait dev` reads `strait.json`, opens a Cloudflare Quick Tunnel, updates each
configured job endpoint to the tunnel URL, and restores the previous endpoints
on shutdown.

```bash
strait dev --file ./strait.json
strait dev --keep-endpoint
```

This is only local endpoint orchestration. Strait does not run your process,
start Docker dependencies, or deploy your runtime.
