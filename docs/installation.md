# Installation

## From source

```bash
go install github.com/strait-dev/cli/cmd/strait@latest
```

Requires Go 1.26 or later.

## Pre-built binaries

Download from [GitHub Releases](https://github.com/strait-dev/cli/releases). Archives are available for Linux, macOS, and Windows on amd64 and arm64.

## Homebrew

```bash
brew install strait-dev/tap/strait
```

## Verify installation

```bash
strait version
```
