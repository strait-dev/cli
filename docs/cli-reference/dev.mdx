## strait dev

Launch a Cloudflare Quick Tunnel to your local serve handler, read
`strait.json`, and temporarily update each job's `endpoint_url` to point at the
tunnel.

```bash
strait dev
strait dev --port 9090
strait dev --file ./strait.json
strait dev --keep-endpoint
```

| Flag | Description |
|---|---|
| `--port` | Local port that the serve handler is listening on |
| `--dir` | Project root containing `strait.json` |
| `--file`, `-f` | Explicit path to a `strait.json` file |
| `--keep-endpoint` | Do not restore previous endpoint URLs on shutdown |

`dev` does not start Docker or deploy customer code. Run your app locally with
your framework's normal command, then use `strait dev` to route signed pushes to
that local process.
