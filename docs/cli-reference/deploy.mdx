## strait deploy

Deploy a managed job image or manifest.

```bash
strait deploy --job my-job --dockerfile ./Dockerfile
strait deploy --config strait.config.json
strait deploy --config strait.config.json --strategy canary --canary-percent 10
```

| Flag | Description | Default |
|---|---|---|
| `--job` | Job slug for single-job deploy | |
| `--config` | Config file for manifest deploy | |
| `--dockerfile` | Path to Dockerfile | `./Dockerfile` |
| `--tag` | Image tag | git SHA |
| `--registry` | Container registry | `registry.fly.io` |
| `--image` | Pre-built image URI (skip build) | |
| `--region` | Region override | |
| `--preset` | Machine preset override | |
| `--env` | Deployment environment | `production` |
| `--strategy` | `direct` or `canary` | `direct` |
| `--canary-percent` | Canary traffic percentage (1-99) | |
| `--canary-duration` | Canary duration (e.g. `10m`) | |
| `--build-arg` | Docker build args (repeatable) | |
| `--cache` | Enable Docker layer caching | `true` |
| `--push` | Push image after build | `true` |
| `--dry-run` | Print plan without executing | `false` |
| `--artifact-uri` | Pre-built artifact URI | |

## strait deploy create

Create a draft deployment.

```bash
strait deploy create --config strait.config.json --artifact-uri registry.example.com/app:v1.0
```

## strait deploy finalize

Finalize a draft deployment.

```bash
strait deploy finalize <deployment-id>
```

## strait deploy promote

Promote a deployment.

```bash
strait deploy promote <deployment-id>
```

## strait deploy rollback

Rollback to a previous deployment.

```bash
strait deploy rollback --to <deployment-id>
```

## strait deploy list

List deployments for a project.

```bash
strait deploy list --project proj_1
```
