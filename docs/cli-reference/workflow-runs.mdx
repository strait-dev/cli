---
title: "workflow-runs"
description: "Manage and inspect workflow run executions"
---

## strait workflow-runs list

List workflow runs for a project.

```bash
strait workflow-runs list --project proj_1
```

| Flag | Description |
|---|---|
| `--project` | Project ID |

## strait workflow-runs get

Get a workflow run by ID.

```bash
strait workflow-runs get <workflow-run-id>
```

## strait workflow-runs steps

Show step-by-step status for a workflow run.

```bash
strait workflow-runs steps list <workflow-run-id>
strait workflow-runs steps approve <workflow-run-id> <step-ref>
strait workflow-runs steps retry <workflow-run-id> <step-ref>
strait workflow-runs steps skip <workflow-run-id> <step-ref>
strait workflow-runs steps force-complete <workflow-run-id> <step-ref> --yes
```

## strait workflow-runs watch

Watch a workflow run until it reaches a terminal state.

```bash
strait workflow-runs watch <workflow-run-id>
strait workflow-runs watch <workflow-run-id> --interval 5s --timeout 10m
```

| Flag | Description | Default |
|---|---|---|
| `--interval` | Poll interval | `2s` |
| `--timeout` | Max watch duration | `5m` |

## strait workflow-runs cancel

Cancel a workflow run.

```bash
strait workflow-runs cancel <workflow-run-id>
```

## strait workflow-runs pause/resume/retry

Pause, resume, or retry a workflow run.

```bash
strait workflow-runs pause <workflow-run-id>
strait workflow-runs resume <workflow-run-id>
strait workflow-runs retry <workflow-run-id>
```

## strait workflow-runs step shortcuts

Top-level shortcuts for individual step actions.

```bash
strait workflow-runs approve-step <workflow-run-id> <step-ref>
strait workflow-runs retry-step <workflow-run-id> <step-ref>
strait workflow-runs skip-step <workflow-run-id> <step-ref>
strait workflow-runs force-complete-step <workflow-run-id> <step-ref> --yes
```
