---
title: "secrets"
description: "Manage server-side and local secrets"
---

## strait secrets list

List server-side secrets for a project.

```bash
strait secrets list --project proj_1
```

## strait secrets create

Create a new secret.

```bash
strait secrets create --project proj_1 --name STRIPE_KEY --value sk_live_xxx
```

## strait secrets delete

Delete a secret.

```bash
strait secrets delete <secret-id>
```

## strait secrets local set

Store a secret in the local system keyring.

```bash
strait secrets local set <name>
```

## strait secrets local get

Retrieve a secret from the local system keyring.

```bash
strait secrets local get <name>
```
