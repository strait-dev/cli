## strait auth login

Authenticate with the Strait API.

```bash
strait auth login                          # Browser-based device code flow
strait auth login --token <api-key>        # Direct token
echo $KEY | strait auth login --with-token # Read from stdin
strait auth login --browser                # Open dashboard to create key
strait auth login --no-browser             # Skip browser, manual entry
```

| Flag | Description |
|---|---|
| `--token` | API key for direct authentication |
| `--with-token` | Read API key from stdin |
| `--api-key` | API key (deprecated, use `--token`) |
| `--context` | Context to save API key under |
| `--server` | Server URL to validate against |
| `--browser` | Open dashboard API key page |
| `--no-browser` | Do not open browser |

## strait logout

Remove stored API key from the system keyring.

```bash
strait logout
strait logout --context staging
```

| Flag | Description |
|---|---|
| `--context` | Context to remove API key from |

## strait auth whoami

Show the authenticated user, active context, server, and project.

```bash
strait auth whoami
```

## strait auth status

Show current authentication state.

```bash
strait auth status
```
