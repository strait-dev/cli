## strait jobs list

List all jobs in a project.

```bash
strait jobs list --project proj_1
```

| Flag | Description |
|---|---|
| `--project` | Project ID |

## strait jobs get

Get a job by ID or slug.

```bash
strait jobs get <job-id-or-slug>
```

## strait jobs create

Create a new job via flags, JSON stdin, or interactive wizard.

<CodeGroup>
```bash Flags
strait jobs create --name "My Job" --slug my-job --endpoint https://example.com/api/process --project proj_1
```

```bash Interactive wizard (TTY)
strait jobs create
```

```bash JSON from stdin
echo '{"name":"my-job","endpoint_url":"http://localhost:3000"}' | strait jobs create --json
```
</CodeGroup>

| Flag | Description | Default |
|---|---|---|
| `--project` | Project ID | from config |
| `--name` | Job name | required (or wizard) |
| `--slug` | Job slug | auto-generated from name |
| `--endpoint` | Endpoint URL | required |
| `--cron` | Cron schedule expression | |
| `--timeout` | Timeout in seconds | `60` |
| `--max-attempts` | Maximum retry attempts | `3` |
| `--json` | Read job definition from stdin as JSON | `false` |

<Info>
  `strait jobs create` also works as an alias for `strait create job` with the same flags (except `--description` instead of wizard, and `--timeout-secs`/`--run-ttl-secs` naming).
</Info>

## strait jobs trigger

Trigger a job run.

```bash
strait jobs trigger <job-id-or-slug> --payload '{"key": "value"}'
```

| Flag | Description |
|---|---|
| `--payload` | Inline JSON payload |
| `--payload-file` | Path to payload JSON file |
| `--priority` | Run priority (default: 0) |
| `--scheduled-at` | RFC3339 timestamp for delayed execution |
| `--idempotency-key` | Prevents duplicate triggers |

<Info>
  `strait trigger <slug>` is a top-level shortcut for `strait jobs trigger`.
</Info>

## strait jobs clone

Clone an existing job.

```bash
strait jobs clone <job-id-or-slug> --slug cloned-job
```

## strait jobs health

Show health metrics for a job.

```bash
strait jobs health <job-id-or-slug>
```

## strait jobs dependencies/add-dependency

List dependency edges for a job or add a new dependency edge.

```bash
strait jobs dependencies <job-id-or-slug>
strait jobs add-dependency <job-id-or-slug> --depends-on upstream-job --type success
```

## strait jobs batch

Apply multiple job updates in one API call.

```bash
strait jobs batch --from-file batch.json
```

## strait jobs update

Update job fields.

```bash
strait jobs update <job-id-or-slug> --name "Nightly sync"
```

## strait jobs versions

List version history for a job.

```bash
strait jobs versions <job-id-or-slug>
```

## strait jobs delete

Delete a job.

```bash
strait jobs delete <job-id-or-slug> --yes
```

| Flag | Description |
|---|---|
| `--yes` | Confirm deletion without prompt |
