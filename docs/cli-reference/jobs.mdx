## strait jobs list

List all jobs in a project.

```bash
strait jobs list --project proj_1
```

| Flag | Description |
|---|---|
| `--project` | Project ID |

## strait jobs get

Get a job by ID or slug.

```bash
strait jobs get <job-id-or-slug>
```

## strait create job

Create a new job via flags, JSON stdin, or interactive wizard.

<CodeGroup>
```bash Flags
strait create job --name "My Job" --slug my-job --endpoint https://example.com/api/process --project proj_1
```

```bash Interactive wizard (TTY)
strait create job
```

```bash JSON from stdin
echo '{"name":"my-job","endpoint_url":"http://localhost:3000"}' | strait create job --json
```
</CodeGroup>

| Flag | Description | Default |
|---|---|---|
| `--project` | Project ID | from config |
| `--name` | Job name | required (or wizard) |
| `--slug` | Job slug | auto-generated from name |
| `--endpoint` | Endpoint URL | required |
| `--cron` | Cron schedule expression | |
| `--timeout` | Timeout in seconds | `60` |
| `--max-attempts` | Maximum retry attempts | `3` |
| `--json` | Read job definition from stdin as JSON | `false` |

<Info>
  `strait jobs create` also works as an alias for `strait create job` with the same flags (except `--description` instead of wizard, and `--timeout-secs`/`--run-ttl-secs` naming).
</Info>

## strait jobs trigger

Trigger a job run.

```bash
strait jobs trigger <job-id-or-slug> --payload '{"key": "value"}'
```

| Flag | Description |
|---|---|
| `--payload` | Inline JSON payload |
| `--payload-file` | Path to payload JSON file |
| `--priority` | Run priority (default: 0) |
| `--scheduled-at` | RFC3339 timestamp for delayed execution |
| `--idempotency-key` | Prevents duplicate triggers |

<Info>
  `strait trigger <slug>` is a top-level shortcut for `strait jobs trigger`.
</Info>

## strait jobs trigger-bulk

Trigger multiple runs for a job in a single API call.

```bash
strait jobs trigger-bulk <job-id-or-slug> --items-json '[{"payload":{"id":"1"}}]'
strait jobs trigger-bulk <job-id-or-slug> --items-file items.json
```

| Flag | Description |
|---|---|
| `--items-json` | JSON array of bulk trigger items |
| `--items-file` | Path to JSON file with bulk trigger items |

## strait jobs describe

Show rich details and recent runs for a job.

```bash
strait jobs describe <job-id-or-slug>
```

## strait jobs edit

Edit a job via single field update or interactive editor.

```bash
strait jobs edit <job-id-or-slug> --field "cron=*/10 * * * *"
strait jobs edit <job-id-or-slug>                # Opens $EDITOR
strait jobs edit <job-id-or-slug> --editor vim
```

| Flag | Description |
|---|---|
| `--field` | Field update in `key=value` form |
| `--editor` | Editor command for interactive mode |

Supported fields: `name`, `slug`, `description`, `cron`, `endpoint_url`, `enabled`, `max_attempts`, `timeout_secs`, `run_ttl_secs`.

## strait jobs versions

List version history for a job.

```bash
strait jobs versions <job-id-or-slug>
```

## strait jobs delete

Disable and delete a job.

```bash
strait jobs delete <job-id-or-slug> --yes
```

| Flag | Description |
|---|---|
| `--yes` | Confirm deletion without prompt |
